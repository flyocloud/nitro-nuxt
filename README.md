# nitrocms-nuxt3

A Nuxt 3 Plugin for Nitro CMS

```
yarn add @flyodev/nitrocms-nuxt3
```

nuxt.config.js

```
modules: ['@flyodev/nitrocms-nuxt3'],
flyo: {
  token: process.env.TOKEN
}
```

## Local Dev

package.json

```
"dependencies": {
    "@flyodev/nitrocms-nuxt3": "./../nitrocms-nuxt3",
    ...
```
